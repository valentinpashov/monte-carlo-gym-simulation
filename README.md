# monte-carlo-gym-simulation
A web-based Monte Carlo simulation designed to forecast gym profitability by analyzing seasonal customer trends, stochastic revenue variables, and cost structures over a 12-month period.

What it does:
Super accurate: Runs 100,000 spins over 20 rounds so the math is solid.

Realistic: It calculates winter peaks, summer lows, and different costs.

Clean charts: Uses Chart.js to show everything visually.
